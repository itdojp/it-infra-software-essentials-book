# ITインフラエンジニアのためのソフトウェア基礎知識

**YAML, JSON, スクリプト, APIで学ぶ自動化の基本**

---

## 本書について

現代のITインフラは、もはや物理的なサーバーやネットワーク機器の導入・保守といった伝統的な業務だけでは完結しません。クラウドコンピューティングの普及、DevOpsの浸透、そしてInfrastructure as Code (IaC) の台頭は、インフラエンジニアの役割と求められるスキルセットを劇的に変化させています。

本書は、ITインフラエンジニアが現代のインフラ運用に必要なソフトウェアの基礎知識を、効率的かつ実践的に習得することを目的としています。

## 対象読者

- LinuxやWindows ServerなどのOS、基本的なネットワークの知識を持つインフラエンジニア
- これからクラウド技術や自動化を学びたいと考えている方
- DevOpsやSREの考え方を自身の業務に取り入れたいと考えている方

## 到達目標

- JSON、YAML形式の設定ファイルやデータ構造を正確に理解し、読み書きができる
- Pythonやシェルスクリプトを用いて、簡単なインフラ自動化スクリプトを作成できる
- RESTful APIの概念を理解し、クラウドサービスとの連携ができる
- Gitを使ってインフラ関連の設定ファイルやスクリプトをバージョン管理できる
- ログ解析や文字列処理に役立つ正規表現の基礎を理解し、実務で活用できる

## 章構成

### [第1章：インフラエンジニアとソフトウェアの接点](src/chapters/chapter01/)
現代インフラにおけるソフトウェア知識の重要性、DevOps、SRE、Infrastructure as Codeの概念を学びます。

### [第2章：データ記述言語の基本と実践](src/chapters/chapter02/) 
JSON、YAMLを中心としたデータ形式の理解と、Pythonでの読み書き方法を実践的に学びます。

### [第3章：スクリプト言語による自動化の初歩](src/chapters/chapter03/)
シェルスクリプトとPythonを使った基本的な自動化スクリプトの作成方法を学びます。

### [第4章：APIによるインフラ連携](src/chapters/chapter04/)
RESTful API の概念を理解し、Pythonでクラウドサービスや外部ツールと連携する方法を学びます。

### [第5章：インフラ運用を支えるその他のソフトウェア知識](src/chapters/chapter05/)
Git、正規表現、データ構造など、インフラ運用に欠かせない基礎知識を習得します。
